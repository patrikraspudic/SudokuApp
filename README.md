# Sudoku App - RAMU 2025/26

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start -c

In order to start the app, Expo Go application needs to be installed on your device.
Your development environment and your mobile phone need to be on the same network.

Application resembles a simple sudoku game, with local leaderboard and in-app time tracker.
